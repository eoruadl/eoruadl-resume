## 프로젝트 개요

Kotlin과 Spring Boot를 중심으로 백엔드 시스템을 고도화하고, 테스트 주도 개발(TDD) 방식으로 품질을 높인 프로젝트입니다.

## 주요 역할 및 구현 사항

- Kotlin + Spring Boot 기반 비즈니스 로직과 REST API 구현
- 테스트 더블(Mock/Stub) 활용으로 단위 테스트 독립성 확보
- 도메인 계층 분리와 서비스 인터페이스 정리로 유지보수성 향상
- Git Workflow 및 PR 리뷰 기준 정리

## 성과

- 회귀 버그 발생 빈도를 줄이고 배포 안정성을 개선했습니다.
- 테스트 코드 기반 리팩터링 속도를 높여 개발 리드타임을 단축했습니다.
