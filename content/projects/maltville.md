## 프로젝트 개요

사용자가 자신만의 위스키 테이스팅 노트를 작성하고 데이터베이스에 안전하게 기록할 수 있는 웹 애플리케이션입니다.

## 주요 역할 및 구현 사항

- Next.js 및 React를 활용한 프론트엔드 로그인 화면과 대시보드 UI 구현
- FastAPI 기반 REST API 설계 및 계층형 서버 구조 적용
- 사용자 인증/세션 관리 구현
- 프로젝트 초기부터 구조화된 API 문서화와 코드 리뷰 프로세스 운영

## 성과

- 주요 사용 시나리오 기준 이탈 구간을 줄이기 위해 화면 흐름을 개선했습니다.
- 백엔드 API 응답 구조를 표준화해 프론트엔드 개발 속도를 높였습니다.
